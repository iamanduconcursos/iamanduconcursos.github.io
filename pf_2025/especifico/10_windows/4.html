Resumo Rápido: Registro do Windows

Disciplina: Sistema Operacional Windows | Concurso: Perito Digital - Polícia Federal

1. O que é o Registro do Windows?

O Registro do Windows é um banco de dados hierárquico que armazena configurações e opções do sistema operacional, aplicativos, serviços e dispositivos de hardware.

  • Substituiu arquivos INI no Windows.
  • Utilizado pelo kernel, drivers e aplicações.
  • Editável via Regedit (regedit.exe) ou Regedt32.
Cuidado: alterações incorretas podem comprometer o sistema.

2. Estrutura do Registro

O registro é dividido em 5 chaves principais (também chamadas de hives):

  • HKEY_CLASSES_ROOT (HKCR): associações de arquivos, extensões e tipos de objeto.
  • HKEY_CURRENT_USER (HKCU): preferências do usuário atualmente logado.
  • HKEY_LOCAL_MACHINE (HKLM): configurações globais do sistema, drivers e serviços.
  • HKEY_USERS (HKU): perfis de todos os usuários do sistema.
  • HKEY_CURRENT_CONFIG (HKCC): informações de hardware da sessão atual.
Mais cobrado em prova: HKLM, HKCU e HKCR.

3. Formato e Funcionamento

  • Cada chave possui subchaves e valores.
  • Valores podem ser do tipo:
    • REG_SZ: String (texto).
    • REG_DWORD: Valor binário de 32 bits.
    • REG_BINARY: Dados binários puros.
  • Armazenado fisicamente em arquivos hive dentro de:
    • %SystemRoot%\System32\Config
    • %UserProfile%\NTUSER.DAT

4. Ferramentas Relacionadas

  • regedit.exe: Interface gráfica para editar o registro.
  • reg.exe: Ferramenta de linha de comando para automação.
  • PowerShell: Acesso avançado ao registro com comandos como Get-ItemProperty e Set-ItemProperty.
  • Políticas de Grupo (GPO): Alterações em massa nas chaves de registro.
