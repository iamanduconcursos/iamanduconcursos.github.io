Resumo: Sistemas Windows 10/11 e Server 2019/2022

Preparação focada para o concurso de Perito Digital da Polícia Federal

1. Fundamentos Comuns

  • Interface Gráfica: Área de trabalho, barra de tarefas, menu Iniciar, Cortana (Windows 10) e Widgets (Windows 11).
  • Gerenciamento de Janelas: Snap Assist, Task View, múltiplos desktops virtuais.
  • Explorador de Arquivos: Navegação, bibliotecas, OneDrive, indexação de arquivos.
  • Segurança: Controle de Conta de Usuário (UAC), Windows Defender, BitLocker (Windows 10/11 Pro).

2. Windows 10

  • Versões: Home, Pro, Enterprise, Education.
  • Instalação: Assistente de instalação, mídia de instalação, atualização via Windows Update.
  • Gerenciamento de Dispositivos: Gerenciador de Dispositivos, drivers, atualizações automáticas.
  • Recursos de Recuperação: Restauração do sistema, redefinir este PC, mídia de recuperação.

3. Windows 11

  • Novidades: Design centrado, novos ícones, integração com Microsoft Teams, suporte a aplicativos Android (via Amazon Store).
  • Requisitos Mínimos: Processador de 64 bits, 4 GB de RAM, 64 GB de armazenamento, TPM 2.0.
  • Gerenciamento de Atualizações: Windows Update, atualizações de recursos e qualidade.

4. Windows Server 2019/2022

  • Funções Principais: Active Directory, DNS, DHCP, Hyper-V, File Services, IIS.
  • Gerenciamento: Server Manager, PowerShell, Remote Server Administration Tools (RSAT).
  • Segurança: Windows Defender, Group Policy, BitLocker, Windows Admin Center.
  • Virtualização: Hyper-V, Containers, Windows Sandbox.
Observação Importante: Em provas da CEBRASPE/CESPE, é comum a cobrança de conhecimentos práticos, como comandos do PowerShell, gerenciamento de permissões NTFS, configuração de políticas de segurança e diagnóstico de problemas comuns em sistemas Windows.

5. Dicas para Provas

  • Prática: Familiarize-se com o ambiente dos sistemas operacionais, especialmente com ferramentas administrativas e de segurança.
  • Comandos Importantes: gpedit.msc, secpol.msc, diskpart, get-help no PowerShell.
  • Segurança: Entenda as configurações de UAC, permissões NTFS, criptografia de arquivos e políticas de senha.
  • Diagnóstico: Saiba como acessar logs de eventos, utilizar o Visualizador de Eventos e interpretar mensagens de erro comuns.
