🖧 Redes de Computadores: Elementos de Interconexão

Resumo essencial e direto ao ponto para o concurso de Perito Digital da PF. Foco nos dispositivos de rede mais cobrados e suas funções nas camadas do modelo OSI.

🔌 1. Tabela Resumo: Dispositivos de Interconexão

Dispositivo Função Camada OSI Observações Importantes
Repetidor Regenera sinais para cobrir maiores distâncias 1 - Física Não interpreta pacotes; atua apenas no sinal
Hub Distribui o sinal para todas as portas 1 - Física Sem inteligência; compartilha largura de banda
Bridge Filtra tráfego entre segmentos de rede 2 - Enlace Utiliza MAC address para tomar decisões
Switch Encaminha quadros baseado no MAC de destino 2 - Enlace Mais eficiente que bridge; comuta tráfego individualmente
Roteador Interliga redes diferentes; encaminha pacotes 3 - Rede Usa endereçamento IP; essencial na Internet
Gateway Converte protocolos entre redes diferentes 7 - Aplicação (ou todas) Permite comunicação entre arquiteturas distintas

📘 2. Detalhamento Estratégico dos Dispositivos

🔁 Repetidor

  • Regenera sinais elétricos ou ópticos enfraquecidos.
  • Utilizado para estender o alcance físico de uma rede.
  • Não toma decisões de encaminhamento.

🔌 Hub

  • Dispositivo de conexão central para redes pequenas.
  • Envia todos os dados recebidos para todas as portas (broadcast).
  • Sem filtragem ou roteamento.
⚠️ CESPE adora: confundir Hub (burro) com Switch (inteligente).

🔀 Bridge

  • Filtra e encaminha quadros entre segmentos de rede com base no endereço MAC.
  • Trabalha na Camada 2 (Enlace).
  • Divide uma rede em domínios de colisão menores.

🎛️ Switch

  • Dispositivo de comutação que envia os dados apenas para a porta de destino.
  • Opera na Camada 2, mas existem switches multicamadas (Camada 3).
  • Aprende e armazena os endereços MAC em uma tabela interna (CAM table).
⚙️ Exemplo de uso: Um switch com 24 portas pode conectar 24 dispositivos que comunicam simultaneamente.

🌐 Roteador

  • Encaminha pacotes entre redes diferentes usando endereços IP.
  • Baseia-se em tabelas de roteamento e protocolos como OSPF, BGP.
  • Fundamental na Internet.
📌 Lembre-se: O roteador separa domínios de broadcast.

🔁 Gateway

  • Dispositivo ou software que converte protocolos entre redes distintas (ex: IP <-> AppleTalk).
  • Pode operar em todas as camadas do modelo OSI (muitas vezes associado à camada 7).
  • Exemplos: gateway de email, gateway de VoIP.
⚠️ Dica CESPE: Gateway ≠ Roteador. Gateway faz conversão de protocolos, o roteador apenas encaminha pacotes IP.

🧠 3. Dicas CESPE/Cebraspe - Como Eles Cobrariam

  • ⚡ Repetidor ≠ Hub: repetidor amplifica, hub apenas repassa.
  • 🔄 Bridge e Switch: mesma camada (2), mas switch é mais moderno e eficiente.
  • 🌍 Gateway: só ele faz tradução entre arquiteturas/protocolos diferentes.
  • 📶 Roteador: único da lista que usa endereços IP (Camada 3).
