📌 1. Fundamentos da Programação Orientada a Objetos (POO)

🎯 Objetos e Classes

Objeto é uma instância de uma classe. Uma classe é como um molde, enquanto o objeto é uma ocorrência concreta.

Exemplo: Classe: Carro | Objeto: meuCarro = new Carro()
  • Atributos: variáveis internas que armazenam o estado do objeto
  • Métodos: funções definidas dentro da classe, determinam comportamento
  • Encapsulamento: proteção dos dados internos do objeto, geralmente com private
Dica CESPE: Questões costumam confundir “classe” e “objeto”. Fique atento ao contexto da pergunta!

🏛️ Herança

Permite que uma classe herde atributos e métodos de outra classe.

  • Classe base (ou superclasse)
  • Classe derivada (ou subclasse)
Exemplo: class Animal → class Cachorro extends Animal
A herança evita repetição de código e promove reuso — conceito que as bancas valorizam como “boas práticas de projeto”.

🧬 Polimorfismo

Capacidade de usar métodos com o mesmo nome, mas comportamentos diferentes conforme o contexto.

  • Em tempo de compilação: Sobrecarga de métodos
  • Em tempo de execução: Sobrescrita (override)
Exemplo: Um método falar() pode emitir sons diferentes em Gato e Cachorro
Atenção: polimorfismo != sobrecarga. A banca adora misturar esses termos em pegadinhas!

➕ Sobrecarga de Métodos

Mesmo nome, diferentes parâmetros dentro da mesma classe.

Exemplo:
void imprimir(String texto)
void imprimir(int numero)
  • É resolvida em tempo de compilação
  • Não tem relação com herança direta

🧠 2. Dicas de Memorização e Revisão Rápida

  • POO = Classe + Objeto + Herança + Polimorfismo + Encapsulamento + Abstração
  • “IS-A” → Herança | “HAS-A” → Composição
  • private, protected, public: controle de acesso
  • Override ≠ Overload
CESPE adora: questões com código e alternativas como “O programa acima imprimirá...” — treine a leitura de código orientado a objetos!

💡 3. Linguagens Mais Cobradas

  • Java: estrutura clássica de POO. Favorita da banca.
  • Python: cobrança recente, foco em sintaxe clara.
  • C++: às vezes aparece em questões mais pesadas.
Priorize estudar Java e Python. C++ costuma ser usado para questões mais conceituais.
