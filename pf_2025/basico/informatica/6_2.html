Informática - Noções de Bancos de Dados

Dados Estruturados e Não Estruturados

1. Definições Básicas

  • Banco de Dados: Conjunto organizado de dados armazenados eletronicamente para fácil acesso, gerenciamento e atualização.
  • Dados Estruturados: Dados organizados em formato fixo, geralmente tabelas, com linhas e colunas bem definidas.
  • Dados Não Estruturados: Dados que não possuem uma organização rígida, não seguem esquema predefinido.

2. Dados Estruturados

  • Formato: tabelas relacionais, com campos e tipos de dados definidos (ex.: SQL).
  • Exemplos: registros bancários, planilhas, cadastros, informações organizadas em colunas e linhas.
  • Vantagens:
    • Fácil consulta e manipulação via linguagens estruturadas (SQL).
    • Integridade e consistência garantidas por regras e restrições.
  • Utilizados em sistemas tradicionais e corporativos.

3. Dados Não Estruturados

  • Formato: arquivos de texto, imagens, vídeos, áudios, emails, documentos, redes sociais.
  • Não possuem formato fixo nem modelo relacional.
  • Difícil processamento por bancos tradicionais, exigem tecnologias específicas (ex.: NoSQL, Big Data).
  • Uso crescente devido ao volume de dados gerados na internet e dispositivos móveis.

4. Comparação entre Dados Estruturados e Não Estruturados

Aspecto Dados Estruturados Dados Não Estruturados
Formato Organizado em tabelas com esquema definido Formato livre, sem esquema definido
Exemplos Bancos relacionais, planilhas, registros Textos, imagens, vídeos, áudios, documentos
Ferramentas SQL, SGBDs relacionais (Oracle, MySQL, SQL Server) NoSQL, Hadoop, MongoDB, ElasticSearch
Complexidade de consulta Alta facilidade com SQL Exige técnicas específicas para busca e análise

Pontos-chave cobrados pela banca CEBRASPE/CESPE:

  • Definição e diferença entre dados estruturados e não estruturados.
  • Exemplos práticos de cada tipo de dado.
  • Características dos bancos de dados relacionais e não relacionais.
  • Vantagens e limitações do uso de dados estruturados vs não estruturados.
  • Contexto atual da explosão de dados não estruturados e tecnologias relacionadas.

Dica final: Foque em entender os conceitos, exemplos e a aplicabilidade dos dados, além de conhecer as ferramentas que lidam com cada tipo. Questões da CEBRASPE valorizam comparação e compreensão prática.
