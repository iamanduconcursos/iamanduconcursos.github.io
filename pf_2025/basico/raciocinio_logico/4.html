Lógica de Primeira Ordem – Resumo para Concurso PF

A lógica de primeira ordem (LPO), também conhecida como lógica de predicados, é uma extensão da lógica proposicional. É mais expressiva e permite representar relações entre objetos e propriedades com o uso de quantificadores.

🎯 CEBRASPE adora testar: interpretação de quantificadores, negação de proposições com ∀ e ∃, e equivalências com predicados.

1. Elementos da Lógica de Primeira Ordem

  • Predicados: Indicam propriedades ou relações entre objetos. Ex: P(x), R(x,y)
  • Variáveis: Símbolos genéricos (x, y, z) que representam elementos do domínio.
  • Quantificadores:
    • Universal (∀): “para todo”, “qualquer que”
    • Existencial (∃): “existe”, “ao menos um”

2. Traduções Comuns (Português ↔ Simbologia)

Português Linguagem Formal
Todos os policiais são honestos ∀x (Policial(x) → Honesto(x))
Alguns servidores são engenheiros ∃x (Servidor(x) ∧ Engenheiro(x))
Nenhum aluno é irresponsável ¬∃x (Aluno(x) ∧ Irresponsável(x))
Se alguém estuda, então será aprovado ∃x (Estuda(x) → Aprovado(x))

3. Negação de Proposições com Quantificadores

Este é o ponto mais cobrado pela banca:

  • ¬(∀x P(x)) ≡ ∃x ¬P(x) (negação do ∀ vira ∃ com negação do predicado)
  • ¬(∃x P(x)) ≡ ∀x ¬P(x) (negação do ∃ vira ∀ com negação do predicado)
Frase: "Todos os sistemas são seguros"
Simbologia: ∀x (Sistema(x) → Seguro(x))
Negação: ∃x (Sistema(x) ∧ ¬Seguro(x))
Tradução: “Existe algum sistema que não é seguro”

4. Implicação com Quantificadores

  • Frases com "todos" geralmente viram uma implicação com quantificador universal: ∀x (P(x) → Q(x))
  • Frases com "existe" são mais comuns com conjunção: ∃x (P(x) ∧ Q(x))

5. Dicas de Ouro para Prova

  • Interprete com calma o que o quantificador está fazendo: abrangência ou existência?
  • Fique atento às negações! O erro mais comum em provas é inverter mal o quantificador.
  • Lembre-se:
    • Negar ∀ é usar ∃ com negação do predicado;
    • Negar ∃ é usar ∀ com negação do predicado.
💡 Exemplo típico CESPE: A negação de “Todos os sistemas são auditáveis” é “Existe ao menos um sistema que não é auditável”.

6. Exemplo de Questão Estilo CESPE

Enunciado: A negação da proposição “Todos os arquivos são criptografados” é logicamente equivalente a:

  1. Nenhum arquivo é criptografado.
  2. Alguns arquivos não são criptografados.
  3. Todos os arquivos não são criptografados.
  4. Se um arquivo é criptografado, então ele não é acessível.

Resposta: Letra B – ∃x (Arquivo(x) ∧ ¬Criptografado(x))
