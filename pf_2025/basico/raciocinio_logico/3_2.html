Tabelas Verdade – Lógica Proposicional

Disciplina: Raciocínio Lógico
Assunto: Lógica Sentencial – Tabelas Verdade
Concurso: Perito Digital da Polícia Federal
Banca: CEBRASPE/CESPE

1. O que é Tabela-Verdade?

É uma ferramenta que permite verificar os valores lógicos (V ou F) de proposições compostas com base nas proposições simples que a compõem.

Usada para:

  • Identificar contradições (sempre falsa)
  • Identificar tautologias (sempre verdadeira)
  • Avaliar a validade de argumentos

2. Conectivos Lógicos e Suas Tabelas

Conjunção ( ∧ ) – “e”

Verdadeira somente se ambas forem verdadeiras.

PQP ∧ Q
VVV
VFF
FVF
FFF

Disjunção ( ∨ ) – “ou”

Falsa somente se ambas forem falsas.

PQP ∨ Q
VVV
VFV
FVV
FFF

Condicional ( → ) – “se... então”

Falsa somente se P é V e Q é F.

PQP → Q
VVV
VFF
FVV
FFV

Bicondicional ( ↔ ) – “se e somente se”

Verdadeira quando ambos os valores são iguais.

PQP ↔ Q
VVV
VFF
FVF
FFV

Negação ( ¬ ) – “não”

Inverte o valor lógico da proposição.

P¬P
VF
FV

3. Construção da Tabela-Verdade Completa

Para N proposições simples, teremos 2ⁿ linhas. Exemplo com 3 proposições (P, Q, R):

2³ = 8 linhas

Monte as combinações V/F de forma binária e aplique as colunas passo a passo.

4. Análise de Argumentos Lógicos

  • Um argumento é válido se, sempre que as premissas forem verdadeiras, a conclusão também for.
  • Use tabela-verdade para testar todos os cenários.
  • Verifique se há alguma linha com premissas V e conclusão F → argumento inválido.

5. Tautologia, Contradição e Contingência

  • Tautologia: sempre verdadeira
  • Contradição: sempre falsa
  • Contingente: varia entre V e F

Exemplo: P ∨ ¬P → Tautologia

Exemplo: P ∧ ¬P → Contradição

6. Dicas para a Prova

✅ Treine lógica com frases comuns. A CEBRASPE adora transformar enunciados simples em estruturas formais!
✅ Use tabelas-verdade reduzidas quando o número de proposições for grande.
⚠️ Erro clássico: Achar que P → Q é falso quando P e Q são falsos.
Na verdade: só é falso quando P é V e Q é F.
